LIBRARY ATTENDANCE SYSTEM - SETUP & USER GUIDE
==============================================

=== PART 1: INITIAL SETUP ===
1. Install Python (3.10 or higher).
2. Open a terminal in this folder.
3. Create a virtual environment:
   python -m venv .venv
4. Activate the environment:
   .venv\Scripts\activate
5. Install dependencies:
   pip install -r requirements.txt
6. Setup the database:
   python manage.py migrate
7. Prepare static files (CSS/Logos):
   python manage.py collectstatic
8. Create an admin account:
   python manage.py createsuperuser

=== PART 2: RUNNING THE SYSTEM ===
Option A: Production Mode (Recommended for Library)
   python run_production.py

Option B: Development Mode
   python manage.py runserver

=== PART 3: CONNECTING MOBILE PHONES (CAMERA) ===
1. Download Ngrok and place `ngrok.exe` in this folder.
2. Open a NEW terminal window.
3. Run: ngrok http 8000
4. Copy the "Forwarding" link (https://....ngrok-free.app).
5. Open that link on your phone to scan QR codes.

=== PART 4: AUTOMATIC CHECKOUT ===
The system is designed to automatically check out users at 5:00 PM daily.

To run this manually (e.g., if the server was off):
   python manage.py force_checkout

=== PART 5: BULK IMPORT CSV FORMAT ===
When uploading students via CSV, ensure the file has these headers:
   - Code (ID Number)
   - First Name
   - Middle Name
   - Last Name
   - Course (e.g., BSCS, Grade 11)
   - Year (e.g., 1, 2, 11)
   - Email

* Note: The system automatically converts acronyms (like 'BSCS') into full names.
* Note: For Basic Ed, 'Course' can be the Grade level.
